型別

Javascript可操作的重要型別是"Object(物件)" 和 "arrays(陣列)"

物件

物件是「名稱/值」對組的群集，或是字串對應至值得map(映射) 

ex: var book = {
		topic:"JavaScript",
		fat:true
	};

book.auther="flangan";
物件.特性=值

book["fat"]=true;
物件["特性"]=boolean;

陣列

ex: var primes = [2,3,5,7];
primes.length = 4;

ex: var points = [{x:0,y:0},{x:1,y:1}];
雙元素的陣列，每個元素都是物件

ex: var data = {
		trial1:[1,2],[3,4],
		trial2:[2,3],[4,5]
	};
雙特性的物件，每個特性都是陣列，陣列中的元素也是陣列

如果Javascript的片語是運算試，完整的句子就稱為述句(Statement)

函式

函式(function)是具有名稱與參數的JavaScript程式碼區塊，只要定義過一次，就可以重複調用(函式本身就是個物件)

當我們將函式與物件結合，就得到方法(methods) 

ex: var a=[];
	a.push(1,2,3);
用push()方法新增一個元素至陣列

換句話說，當函式被指定給某個物件當特性，就稱他們為方法

名詞

  • excution context(執行上下文 執行區域)
  • outer enviroment(外部環境)
  • Scope Chain(區域鍊)
  • asynchronous(異步)
  • event queue(事件列隊)
  • Precedence(優先)
  • Associativity(關聯)
  • coercion(強迫 強轉型態)
  • IIFEs(Immediately Invoked Functions Expressions)

簡單的來說，Scope Chain就是ㄉ在找變項的過程中，從內層找到外層，直到最外面的global environment的這條鍊，就叫做Scope Chain。按照上面的例子來說，就是先從function b()開始，在function b()裡面找myVar這個變項，可是找不到，因此便繼續往外層找，在最外層的global environment中，找到了myVar = 1。

JavaScript Engine會先把execution context的內容執行完畢，接著才去執行"event queue"

IIFEs(Immediately Invoked Functions Expressions) 其實可以就字面上直接來理解，Immediately就是立即的意思，invoked則是執行某個函式時，「執行」的意思，function expression是一種用來建立function的方法，總括來說，就是用function expression的方式建立函式後並立即執行它。
